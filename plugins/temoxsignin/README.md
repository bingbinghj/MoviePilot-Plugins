# 中国特摄联盟自动登录

MoviePilot V2 插件，用于每天自动登录中国特摄联盟：

```text
http://bt.temox.com:8080
```

## 功能

- 支持站点访问前置算术验证，例如 `3 + 8 + 7 = ?`
- 自动解析 Discuz X3.4 登录表单并提交账号密码
- 支持 Cron 定时每天执行
- 支持配置页手动执行一次
- 支持远程命令 `/temox_signin`

## 配置

- `启用插件`：开启后注册定时任务
- `站点地址`：默认 `http://bt.temox.com:8080`
- `用户名` / `密码`：中国特摄联盟账号密码
- `Cron 表达式`：默认 `15 8 * * *`，即每天 08:15 执行
- `仅运行一次`：保存配置后立即执行一次，并自动关闭该开关
- `发送通知`：执行结束后尝试通过 MoviePilot 通知渠道发送结果

如果账号设置了 Discuz 安全提问，可填写安全提问编号和答案；未设置时保持默认 `0` 即可。
